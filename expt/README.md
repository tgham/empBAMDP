# An integrative framework for the human sense of control

This repository contains the experimental code (subfolders `Experiment1`, `Experiment2`, and `Experiment3`), experimental data (subfolder `EmpHCA/data`), and the analysis code (subfolder `EmpHCA`) for the results reported in the article:

Alireza Modirshanechi, Peter Dayan*, and Eric Schulz*, [“An integrative framework for the human sense of control”](https://osf.io/cnkyz), PsyArXiv. (2025)

Each subfolder contains its own dedicated `README.md`.
